# Brazilian E-Commerce Analysis

End-to-end analytics project based on the 
[Olist Brazilian E-Commerce dataset](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce).

## Project Goal
Analyze Brazilian e-commerce data to identify revenue drivers, 
delivery performance issues, customer behavior patterns, 
and provide actionable business recommendations.

## Tools Used
- **Python** (pandas, matplotlib, seaborn, scipy) — data cleaning, analysis, visualizations
- **BigQuery** — SQL queries for business questions
- **Tableau Public** — interactive dashboards

## Dataset
- Source: [Kaggle — Olist Brazilian E-Commerce](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce)
- 99,441 orders, 8 tables
- Period: September 2016 — August 2018

## Project Structure
- `notebook/` — Jupyter notebook with full analysis
- `sql/` — BigQuery SQL queries
- `visualizations/` — exported charts

## Key Findings
- **health_beauty** is the top revenue category (9.4% of total)
- **São Paulo** generates 38% of total revenue
- Late deliveries have significantly lower review scores (2.26 vs 4.21)
- Only **3%** repeat purchase rate — low customer retention
- **Credit card** dominates payments (74% of transactions)
